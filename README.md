# Portfolio

Projeto base com `Next.js`, `React` e `Tailwind CSS`, pronto para evoluir do zero.

## Requisitos

- Node `20.9.0` ou superior
- npm `10+`

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Estrutura atual

```text
app/
  globals.css
  layout.tsx
  page.tsx
```

## Observacoes

- A pagina inicial esta propositalmente minima para comecar do zero.
- O projeto usa `App Router`.
